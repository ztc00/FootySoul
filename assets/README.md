# Assets Directory

Place your app assets here:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)
- `notification-icon.png` - Notification icon (96x96px)

For now, placeholder assets are being used. You can generate proper assets later using:
- Expo's asset generator
- Online tools like AppIcon.co
- Design tools like Figma
